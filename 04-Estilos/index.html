Estilos

CSS significa C ascading S tyle S heets.

CSS describe cómo se deben mostrar los elementos HTML en la pantalla, papel u otros medios.

CSS se puede agregar a elementos HTML de 3 maneras:

  • En línea : mediante el uso del atributo de estilo en elementos HTML.
  • Interno : mediante el uso de un style elemento en la head sección.
  • Externo : mediante el uso de un archivo CSS externo.

El atributo de estilo HTML

p style="propiedad:valor;"/p

Valores de color

En HTML, los colores también se pueden especificar utilizando valores RGB, valores HEX, valores HSL, valores RGBA y valores HSLA.

  • Valor RGB

    • En HTML, un color puede especificarse como un valor RGB, utilizando esta fórmula: rgb ( rojo, verde , azul ). Cada parámetro (rojo, verde y azul) define la intensidad del color entre 0 y 255. Para mostrar el negro, configure todos los parámetros de color en 0, Para mostrar el blanco, configure todos los parámetros de color en 255.

    Tomato

  • Valores Grises con RGB

    • Los tonos de gris a menudo se definen utilizando valores iguales para las 3 fuentes de luz

    0

    60

    120

    180

    240

    255

  • Valor Hexadecimal

    • En HTML, se puede especificar un color utilizando un valor hexadecimal en la forma:# rrggbb. Donde rr (rojo), gg (verde) y bb (azul) son valores hexadecimales entre 00 y ff (igual que el decimal 0-255).

    Tomato

  • Valores grises con Hexadecimal

    • Los tonos de gris a menudo se definen utilizando valores iguales para las 3 fuentes de luz.

    #000000

    #3c3c3c

    #787878

    #b4b4b4

    #f0f0f0

    #ffffff

  • Valor HSL

    • En HTML, se puede especificar un color utilizando matiz, saturación y luminosidad (HSL) en la forma: hsl ( tono , saturación , luminosidad ). El tono es un grado en la rueda de color de 0 a 360. 0 es rojo, 120 es verde y 240 es azul. La saturación es un valor porcentual, 0% significa un tono de gris y 100% es el color completo. La claridad también es un porcentaje, 0% es negro, 50% no es claro ni oscuro, 100% es blanco.

    Tomato

  • Saturación

    • La saturación se puede describir como la intensidad de un color. 100% es color puro, sin sombras de gris. 50% es 50% gris, pero aún puede ver el color. 0% es completamente gris, ya no puede ver el color.

    hsl(0, 100%, 50%)

    hsl(0, 80%, 50%)

    hsl(0, 60%, 50%)

    hsl(0, 40%, 50%)

    hsl(0, 20%, 50%)

    hsl(0, 0%, 50%)

  • Ligereza

    • La claridad de un color se puede describir como la cantidad de luz que desea dar al color, donde 0% significa que no hay luz (negro), 50% significa 50% de luz (ni oscura ni clara) 100% significa claridad total (blanco).

    hsl(0, 100%, 0%)

    hsl(0, 100%, 25%)

    hsl(0, 100%, 50%)

    hsl(0, 100%, 75%)

    hsl(0, 100%, 90%)

    hsl(0, 100%, 100%)

  • Valores grises con HSL

    • Los tonos de gris a menudo se definen configurando el tono y la saturación en 0, y ajustando la claridad del 0% al 100% para obtener tonos más oscuros / claros.

    hsl(0, 0%, 0%)

    hsl(0, 0%, 24%)

    hsl(0, 0%, 47%)

    hsl(0, 0%, 71%)

    hsl(0, 0%, 94%)

    hsl(0, 0%, 100%)

  • Valor RGBA

    • Los valores de color RGBA son una extensión de los valores de color RGB con un canal alfa, que especifica la opacidad de un color. El parámetro alfa es un número entre 0.0 (totalmente transparente) y 1.0 (no transparente en absoluto).

    Tomato 50% transparente

  • Valor HSLA

    • Los valores de color HSLA son una extensión de los valores de color HSL con un canal alfa, que especifica la opacidad de un color. El parámetro alfa es un número entre 0.0 (totalmente transparente) y 1.0 (no transparente en absoluto).

    Tomato 50% transparente
