Clases y ID

El atributo de clase

El classatributo HTML se usa para definir estilos iguales para elementos con el mismo nombre de clase. Por lo tanto, todos los elementos HTML con el mismo classatributo obtendrán el mismo estilo.

En CSS, para seleccionar elementos con una clase específica, escriba un carácter de punto (.), Seguido del nombre de la clase.

Diferentes etiquetas pueden tener el mismo nombre de clase y, por lo tanto, compartir el mismo estilo.

London

London es la capital de inglaterra.

Paris

Paris es la capital de Francia.

Tokyo

Tokyo es la capital de Japon.

El class atributo HTML también se puede usar en elementos en línea.

Mi encabezado imporatante.

Este es texto importante.

El class atributo se puede usar en cualquier elemento HTML. El nombre de la clase distingue entre mayúsculas y minúsculas.

Clases Múltiples

Usando el atributo de clase en JavaScript

JavaScript también puede utilizar el nombre de la clase para realizar ciertas tareas para elementos con el nombre de clase especificado.

JavaScript puede acceder a elementos con un nombre de clase específico mediante el getElementsByClassName()método.

El atributo id

El idatributo especifica una identificación única para un elemento HTML (el valor debe ser único dentro del documento HTML).

CSS y JavaScript pueden usar el valor de identificación para realizar ciertas tareas para el elemento con el valor de identificación específico.

En CSS, para seleccionar un elemento con una identificación específica, escriba un carácter hash (#), seguido de la identificación del elemento

El atributo id se puede usar en cualquier elemento HTML. El valor de identificación distingue entre mayúsculas y minúsculas. El valor de identificación debe contener al menos un carácter y no debe contener espacios en blanco.

Diferencia entre clase e identificación

Un elemento HTML solo puede tener una identificación única que pertenezca a ese único elemento, mientras que varios elementos pueden usar un nombre de clase.

Marcadores con ID y enlaces

Los marcadores HTML se utilizan para permitir a los lectores saltar a partes específicas de una página web. Los marcadores pueden ser útiles si su página web es muy larga. Para hacer un marcador, primero debe crear el marcador y luego agregarle un enlace. Cuando se hace clic en el enlace, la página se desplazará a la ubicación con el marcador.

Usando el atributo id en JavaScript

JavaScript puede acceder a un elemento con una identificación especificada mediante el getElementById()método.
