Encabezados HTML

Los encabezados se definen con las etiquetas h1 a h6.

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Cada encabezado HTML tiene un tamaño predeterminado. Sin embargo, puede especificar el tamaño de cualquier encabezado con el style atributo.

Enlaces

Los enlaces permiten a los usuarios hacer clic de una página a otra. Los enlaces HTML son hipervínculos. Cuando mueves el mouse sobre un enlace, la flecha del mouse se convertirá en una pequeña mano.

Nota: Un enlace no tiene que ser texto. Puede ser una imagen o cualquier otro elemento HTML. border:0; se agrega para evitar que IE9 (y versiones anteriores) muestren un borde alrededor de la imagen (cuando la imagen es un enlace).

El target atributo especifica dónde abrir el documento vinculado.

El title atributo especifica información adicional sobre un elemento. La información se muestra con mayor frecuencia como texto de información sobre herramientas cuando el mouse se mueve sobre el elemento.

Listas

Una lista desordenada comienza con la etiqueta ul. Una lista ordenada comienza con la etiqueta ol. Cada elemento de la lista comienza con la etiqueta li.

La list-style-type propiedad CSS se utiliza para definir el estilo del marcador del elemento de la lista desordenada.

Valores posibles:
Valor Descripción
disc Establece el marcador del elemento de la lista en una viñeta (predeterminado).
circle Establece el marcador del elemento de la lista en un círculo.
square Establece el marcador del elemento de la lista en un cuadrado
none Los elementos de la lista no se marcarán.

El type atributo de la etiqueta, define el tipo de marcador de elemento de lista ordenada.

Valores posibles:
Valor Descripción
type="1" Los elementos de la lista estarán numerados con números (predeterminado).
type="A" Los elementos de la lista estarán numerados con letras mayúsculas.
type="a" Los elementos de la lista estarán numerados con letras minúsculas.
type="I" Los elementos de la lista estarán numerados con números romanos en mayúscula.
type="i" Los elementos de la lista estarán numerados con números romanos en minúsculas.

La lista se puede anidar (listas dentro de listas). Los elementos de la lista pueden contener una nueva lista y otros elementos HTML, como imágenes y enlaces, etc.

Por defecto, una lista ordenada comenzará a contar desde 1. Si desea comenzar a contar desde un número especificado, puede usar el startatributo.

Listas de descripción

Una lista de descripción es una lista de términos, con una descripción de cada término.

La dl etiqueta define la lista de descripción, la dt etiqueta define el término (nombre) y la etiqueta dd describe cada término
