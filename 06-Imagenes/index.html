Imagenes

Las imágenes pueden mejorar el diseño y la apariencia de una página web.

El alt atributo es obligatorio. Una página web no se validará correctamente sin ella.

Especifique siempre el ancho y el alto de una imagen. Si no se especifican el ancho y el alto, la página puede parpadear mientras se carga la imagen.

Si no se especifica, el navegador espera encontrar la imagen en la misma carpeta que la página web.

Puede acceder a imágenes desde cualquier dirección web en el mundo.

Cargar imágenes lleva tiempo. Las imágenes grandes pueden ralentizar su página. Usa imágenes con cuidado.

Imágenes animadas

HTML permite GIF animados.

Imagen flotante

Use la floatpropiedad CSS para dejar que la imagen flote a la derecha o a la izquierda de un texto

Mapas de imagen

Un mapa de imagen es una imagen con áreas en las que se puede hacer clic.

El usemap valor comienza con una etiqueta hash #seguida del nombre del mapa de imagen y se usa para crear una relación entre la imagen y el mapa de imagen.

El map elemento se utiliza para crear un mapa de imagen y se vincula a la imagen mediante el name atributo.

Un área en la que se puede hacer clic se define usando un area elemento.

    Debe definir la forma del área y puede elegir uno de estos valores:

  • rect - define una región rectangular
  • circle - define una región circular
  • poly - define una región poligonal
  • default - define toda la región

Coordenadas para rect: 1. distancia del margen izq. 2. distancia del margen superior. 3. distancia del margen izq. 4. distancia del margen superior.

Para agregar un área circular, primero ubique las coordenadas del centro del círculo. Luego especifique el radio del círculo

Mapa de imagen y JavaScript

Un área en la que se puede hacer clic no tiene que ser un enlace a otra página, también puede activar una función de JavaScript.

Imágenes de fondo

Se puede especificar una imagen de fondo en casi cualquier elemento HTML. Para agregar una imagen de fondo en HTML, use la propiedad CSS background-image.

Imagen de fondo en una página

Si desea que toda la página tenga una imagen de fondo, debe especificar la imagen de fondo en el elemento body.

Si la imagen de fondo es más pequeña que el elemento, la imagen se repetirá, horizontal y verticalmente, hasta que llegue al final del elemento.

Para evitar que la imagen de fondo se repita, use la background-repeat: no-repeat propiedad

Cubierta de fondo

Si desea que la imagen de fondo cubra todo el elemento, puede establecer la background-sizepropiedad en cover. Además, para asegurarse de que todo el elemento esté siempre cubierto, establezca la propiedad background-attach en fixed. Esto es más para fondo del body.

Elemento picture

El elemento de imagen nos permite mostrar diferentes imágenes para diferentes dispositivos o tamaños de pantalla.

El elemento picture contiene varios elementos source, cada uno de los cuales se refiere a diferentes fuentes de imagen. De esta manera, el navegador puede elegir la imagen que mejor se adapte a la vista y / o dispositivo actual.

Siempre especifique un img elemento como el último elemento hijo del picture. El img elemento es utilizado por los navegadores que no lo admiten, o si ninguna de lassource etiquetas coincide.

El navegador usará el primer source elemento con valores de atributo coincidentes e ignorará los siguientes.
