Tablas

Una tabla HTML se define con la etiqueta table. Cada fila de la tabla se define con la etiqueta tr. Un encabezado de tabla se define con la etiqueta th. Por defecto, los encabezados de las tablas están en negrita y centrados. Una tabla de datos / celda se define con la etiqueta td.

Los elementos son los contenedores de datos de la tabla. Pueden contener todo tipo de elementos HTML; texto, imágenes, listas, otras tablas, etc.

Si no especifica un borde para la tabla, se mostrará sin bordes. Si desea que los bordes se contraigan en un borde, agregue la border-collapse propiedad CSS. El relleno de celda especifica el espacio entre el contenido de la celda y sus bordes. Si no especifica un relleno, las celdas de la tabla se mostrarán sin relleno.

Para hacer que una celda abarque más de una columna, use el colspan atributo. Para hacer que una celda abarque más de una fila, use el rowspanatributo

Para agregar un título a una tabla, use la caption etiqueta.

Agregar espacio de borde

El espacio entre bordes especifica el espacio entre las celdas. Para establecer el espacio entre bordes para una tabla, use la border-spacing propiedad CSS. Si la tabla tiene bordes colapsados, no tiene efecto.
